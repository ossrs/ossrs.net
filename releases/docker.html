Install Docker

请在Docker官网下载并安装桌面版(DockerDesktop)。

Run SRS 3.0

SRS目前提供的稳定版是3.0，使用Docker官方镜像启动： 

echo "Run SRS 3.0 in docker" &&
    docker run --rm -p 1935:1935 -p 1985:1985 -p 8080:8080 ossrs/srs:3

国内可以使用阿里云镜像，下载速度更快： 

# Or from images at Aliyun
    docker run --rm -p 1935:1935 -p 1985:1985 -p 8080:8080 \
        registry.cn-hangzhou.aliyuncs.com/ossrs/srs:3

SRS每个版本都有对应的Docker镜像，请参考Tags。

Run SRS 4.0

SRS 4.0还没有发布，但已经在打TAG，所以也可以用Docker运行，具体用法请参考这里。

Verify

启动后就可以推流和播放，请参考SRS:Usage， 或者SRS:3.0， 或者SRS:4.0。
